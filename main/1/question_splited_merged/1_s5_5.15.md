Problem 5.15. Show that in special relativity the stress-energy of an isolated physical system of finite extent obeys the tensor virial theorem
$$
\int \mathrm {T} ^ {\mathrm {i j}} \mathrm {d} ^ {3} \mathbf {x} = \frac {1}{2} \frac {\mathrm {d} ^ {2}}{\mathrm {d t} ^ {2}} \int \mathrm {T} ^ {0 0} \mathbf {x} ^ {\mathrm {i}} \mathbf {x} ^ {\mathrm {j}} \mathrm {d} ^ {3} \mathbf {x}.
$$

## Solution

Solution 5.15. The technique here is to use the equations of motion $(\mathbf{T}^{\mu \nu}, \nu = 0)$ to convert first derivative terms to divergences and then, by Gauss's theorem, to vanishing surface integrals outside the system. By repeated use of the equations of motion we have
$$
\mathrm {T} _ {, 0 0} ^ {0 0} = - \mathrm {T} _ {, \mathrm {k} 0} ^ {0 \mathrm {k}} = - \mathrm {T} _ {, 0 \mathrm {k}} ^ { \mathrm {k } 0 } = \mathrm {T} _ {, \mathrm {mk} } ^ { \mathrm {km} }.
$$
Now multiply by $\mathbf{x}^i\mathbf{x}^j$
$$
\begin{array}{l} \left(\mathbf {x} ^ { \mathrm {i}} \mathbf {x} ^ { \mathrm {j}} \mathbf {T} ^ {0 0}\right) _ {, 0 0} = \mathbf {x} ^ { \mathrm {i}} \mathbf {x} ^ { \mathrm {j}} \left(\mathbf {T} ^ { \mathrm {km} }\right) _ {, \mathrm {km} } \\ = \left(\mathrm {T} ^ { \mathrm {km} } \mathrm {x} ^ { \mathrm {i} } \mathrm {x} ^ { \mathrm {j}}\right) _ { \mathrm {,km}} - 2 \left(\mathrm {x} ^ { \mathrm {i} } \mathrm {x} ^ { \mathrm {j}}\right) _ { \mathrm {,k}} \mathrm {T} ^ { \mathrm {km} } _ { \mathrm {,m}} - \mathrm {T} ^ { \mathrm {km} } \left(\mathrm {x} ^ { \mathrm {i} } \mathrm {x} ^ { \mathrm {j}}\right) _ { \mathrm {,km}} \\ = \left(\mathrm {T} ^ { \mathrm {km} } \mathrm {x} ^ { \mathrm {i} } \mathrm {x} ^ { \mathrm {j}}\right) _ { \mathrm {,km}} - 2 \left(\mathrm {x} ^ { \mathrm {j}} \mathrm {T} ^ { \mathrm {im} } _ { \mathrm {,m}} + \mathrm {x} ^ { \mathrm {i} } \mathrm {T} ^ { \mathrm {jm} } _ { \mathrm {,m}}\right) - 2 \mathrm {T} ^ { \mathrm {ij} }. \\ \end{array}
$$
The first derivative terms can be simplified, e.g.
$$
\mathbf {x} ^ { \mathrm {j}} \mathbf {T} _ {, \mathrm {m} } ^ { \mathrm {im} } = (\mathbf {x} ^ { \mathrm {j}} \mathbf {T} ^ { \mathrm {im} }) _ {, \mathrm {m}} - \delta_ { \mathrm {m}} ^ { \mathrm {j} } \mathbf {T} ^ { \mathrm {im} } .
$$
If we make this simplification, integrate over all space, and ignore all divergences which can be converted to vanishing surface integrals we see
$$
\begin{array}{l} \frac {d ^ {2}}{d t ^ {2}} \int x ^ {i} x ^ {j} T ^ {0 0} d ^ {3} x = \int [ 0 - 2 (- T ^ {ij} - T ^ {ij}) - 2 T ^ {ij} ] d ^ {3} x \\ = 2 \int T ^ {ij} d ^ {3} x. \\ \end{array}
$$
