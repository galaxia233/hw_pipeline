Problem 4.15. Show that the stress-energy tensor for the electromagnetic field is divergenceless (i.e. $\mathbf{T}^{\mu \nu}_{,\nu} = 0$ ) in the absence of charge sources.

## Solution

Solution 4.15. The form of the electromagnetic stress-energy is given in the introduction. Taking its divergence we have:
$$
\begin{array}{l} 4 \pi \mathrm {T} ^ { \mu \nu } , _ { \nu } = \mathrm {F} ^ { \mu a } , _ { \nu } \mathrm {F} ^ { \nu } _ { a } + \mathrm {F} ^ { \mu a } \mathrm {F} ^ { \nu } _ { a , \nu } - \frac {1}{2} \mathrm {F} _ { a } \beta \mathrm {F} ^ { \alpha \beta , \mu } \\ = \mathrm {F} ^ { \mu a } \mathrm {F} _ { \alpha , \nu } ^ { \nu } + \mathrm {F} _ { \nu a } \left(\mathrm {F} ^ { \mu a , \nu } - \frac {1}{2} \mathrm {F} ^ { \nu a , \mu }\right) \\ = \mathrm {F} ^ { \mu \alpha } \mathrm {F} _ { \alpha , \nu } ^ { \nu } + \frac {1}{2} \mathrm {F} _ { \nu \alpha } \left(\mathrm {F} ^ { \alpha \nu , \mu } + \mathrm {F} ^ { \mu \alpha , \nu } + \mathrm {F} ^ { \nu \mu , \alpha }\right). \\ \end{array}
$$
According to the free field Maxwell equations, both $\mathbf{F}^{\nu}_{\alpha,\nu}$ and the terms in parentheses vanish, hence $\mathbf{T}^{\mu\nu}_{,\nu} = 0$ . Note: If a charge current $\mathbf{J}$ is present this result is modified to $\mathbf{T}^{\mu\nu}_{,\nu} = -\mathbf{F}^{\mu\alpha}\mathbf{J}_{\alpha}$ , as is easily verified.
