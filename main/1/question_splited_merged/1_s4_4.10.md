Problem 4.10. Write out the $\mu = 0$ component of the Lorentz force equation $\mathrm{du}^{\mu} / \mathrm{d}\tau = (\mathrm{e} / \mathrm{m})\mathrm{F}^{\mu \beta}\mathrm{u}_{\beta}$ expressing $\mathbf{F}^{\mu \nu}$ in terms of $\mathbf{E}_i$ and $\mathbf{B}_i$ , to obtain
$$
\mathrm {d P} ^ {0} / \mathrm {d t} = \underset {\sim} {\mathrm {e}} \underset {\sim} {\mathrm {v}} \cdot \underset {\sim} {\mathrm {E}}.
$$

## Solution

Solution 4.10. With $\mu = 0$ the Lorentz force equation is
$$
\mathrm {d} \mathbf {P} ^ { 0 } / \mathrm {d} \tau = \mathrm {eF} ^ { 0 \mathrm {i} } \mathbf {u} _ { \mathrm {i} } = \mathrm {eE} ^ { \mathrm {i} } \gamma \mathbf {v} _ { \mathrm {i} }.
$$
Since $\mathrm{d}\tau = \mathrm{d}t / \gamma$ this equation can be written
$$
\mathrm {d} \mathbf {P} ^ { 0 } / \mathrm {d} t = e E ^ { i } v _ { i }.
$$
Note that this differs from the nonrelativistic equation only in that $\mathbf{P}^0$ is the relativistic mass-energy.
