4.13 (c) Problem 4.13. A particle of charge $q$ , mass $m$ , moves in a circular orbit of radius $R$ in a uniform B field $\mathbf{B}_{\underline{\mathbf{e}}_z}$ . (c) Calculate $\mathrm{du}^{0^{'}} / \mathrm{dr}$ and thus $\mathrm{dP}^{0^{'}} / \mathrm{d}\tau$ . Explain how the energy of the particle can change since the B field does no work on it.

## Solution

(c) Since $\mathfrak{u}^{0'}$ is not constant, energy is seen to increase:
$$
\frac { \mathrm {d} \mathrm {p} ^ { 0 ^ { \prime } } } { \mathrm {d} r } = \mathrm {m} \frac { \mathrm {d} \mathrm {u} ^ { 0 ^ { \prime } } } { \mathrm {d} r } = - \frac { \mathrm {m} \beta \omega \gamma \mathrm {u} ^ { \mathrm {y} } } { \left( 1 - \omega^ { 2 } \mathrm {R} ^ { 2 } \right) ^ { \frac { 1 } { 2 } } }
$$
In the moving observer's frame he observes an $\underline{\mathbf{E}}$ field $\mathbf{E}^{\mathbf{y}'} = -\gamma \beta \mathbf{B}$ . (This is easily computed by Lorentz transforming $\mathbf{F}^{\mu \nu}$ .) The observer should then expect to see work being done on the particle at a rate
$$
\frac { \mathrm {d} \mathrm {p} ^ { 0 ^ { \prime } } } { \mathrm {d} r } = \mathrm {qE} ^ { \mathrm {y} ^ { \prime } } \mathrm {u} ^ { \mathrm {y} ^ { \prime } } = - \frac { \mathrm {m} \omega \gamma \mathrm {u} ^ { \mathrm {y} ^ { \prime } } } { \left( 1 - \omega^ { 2 } \mathrm {R} ^ { 2 } \right) ^ { \frac { 1 } { 2 } } } = - \frac { \mathrm {m} \omega \gamma \mathrm {u} ^ { \mathrm {y} } } { \left( 1 - \omega^ { 2 } \mathrm {R} ^ { 2 } \right) ^ { \frac { 1 } { 2 } } }
$$
This resolves the "paradox."
