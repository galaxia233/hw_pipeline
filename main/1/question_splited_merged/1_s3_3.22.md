Problem 3.22. Evaluate $\varepsilon_{\mu \nu \rho \sigma} \varepsilon^{\mu \nu \rho \sigma}$ .

## Solution

Solution 3.22. Evaluate this scalar in a local orthonormal frame. According to Solution 3.21, $\varepsilon_{\mu \nu \rho \sigma} = -\varepsilon^{\mu \nu \rho \sigma}$ so
$$
\varepsilon_ { \mu \nu \rho \sigma } \varepsilon^ { \mu \nu \rho \sigma } = - \sum_ { \mu \nu \rho \sigma } | \varepsilon_ { \mu \nu \rho \sigma } | ^ { 2 } = - \sum | \varepsilon_ { 0 1 2 3 } | ^ { 2 }
$$
where the sum is over permutations of 0123. Since there are $4! = 24$ permutations, $\varepsilon_{\mu \nu \rho \sigma}^{\mu \nu \rho \sigma} = -24$ .
