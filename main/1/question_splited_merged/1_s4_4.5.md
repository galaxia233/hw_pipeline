Problem 4.5. Prove that except when $(\underline{\mathbf{B}}\cdot \underline{\mathbf{E}})^2 +(\mathbf{B}^2 -\mathbf{E}^2)^2 = 0$ , there is a Lorentz transformation which will make E and B parallel $(\underline{\mathbf{E}}^{\prime}\times \underline{\mathbf{B}}^{\prime} = 0)$ [Hint: Try $\underline{\mathbf{v}} = a(\underline{\mathbf{E}}\times \underline{\mathbf{B}})$ for some $\alpha .]$

## Solution

Solution 4.5. Let $\underline{\mathbf{v}} = \alpha (\underline{\mathbf{E}}\times \underline{\mathbf{B}})$ , then
$$
\begin{array}{l} \underline { { { \underline { { { \mathbf {E } } } } } } } \dot { \gamma } ^ { - 1 } = \underline { { { \underline { { { \mathbf {E } } } } } } } + \underline { { { \underline { { { \mathbf {v } } } } } } } \times \underline { { { \underline { { { \mathbf {B } } } } } } } = (1 - \alpha \mathbf {B } ^ { 2 } ) \underline { { { \underline { { { \mathbf {E } } } } } } } + \alpha ( \underline { { { \mathbf {E } } } } \cdot \underline { { { \mathbf {B } } } } ) \underline { { { \mathbf {B } } } } \\ \underline { { { \mathbf {B } } } } \dot { \gamma } ^ { - 1 } = \underline { { { \mathbf {B } } } } - \underline { { { \mathbf {v } } } } \times \underline { { { \mathbf {E } } } } = (1 - a \mathbf {E } ^ { 2 } ) \underline { { { \mathbf {B } } } } + a ( \underline { { { \mathbf {E } } } } \cdot \underline { { { \mathbf {B } } } } ) \underline { { { \mathbf {E } } } } \\ \end{array}
$$
Case i) If $\underline{\mathbf{E}}\cdot \underline{\mathbf{B}} = 0$ and $|\mathbf{E}| = |\mathbf{B}|$ , the transformation is equivalent to the redshift of a plane wave, and $\underline{\mathbf{E}}$ and $\underline{\mathbf{B}}$ cannot be made parallel.
Case ii) If $\underline{\mathbf{E}}\cdot \underline{\mathbf{B}} = 0$ but $\mathbf{E}^2\neq \mathbf{B}^2$ choose $a = 1 / \max (\mathbf{E}^2,\mathbf{B}^2)$ to make $\underline{\mathbf{E}}'$ or $\underline{\mathbf{B}}'$ vanish. Then $\underline{\mathbf{E}}^{\prime}\times \underline{\mathbf{B}}^{\prime} = 0$
Case iii) If $\underline{\mathbf{E}}\cdot \underline{\mathbf{B}}\neq 0$ , then $\underline{\mathbf{E}}^{\prime}$ and $\underline{\mathbf{B}}^{\prime}$ can be made parallel by choosing $\alpha$ to satisfy $a^2 [(\underline{\mathbf{E}}\cdot \underline{\mathbf{B}})^2 -\mathbf{E}^2\mathbf{B}^2 ] = 1 - a(\mathbf{E}^2 +\mathbf{B}^2)$ , so that
$$
\frac { \mathbf { \underline { { { v } } } } } { 1 + \mathbf {v } ^ { 2 } } = \frac { a ( \mathbf { \underline { { {E } } } } \times \mathbf { \underline { { {B } } } } ) } { 1 + a ^ { 2 } [ ( \mathbf { \underline { { {E } } } } \times \mathbf { \underline { { {B } } } } ) ] ^ { 2 } } = \frac { a ( \mathbf { \underline { { {E } } } } \times \mathbf { \underline { { {B } } } } ) } { 1 + a ^ { 2 } [ \mathbf {E } ^ { 2 } \mathbf {B } ^ { 2 } - ( \mathbf { \underline { { {E } } } } \cdot \mathbf { \underline { { {B } } } } ) ^ { 2 } ] } = \frac { \mathbf { \underline { { {E } } } } \times \mathbf { \underline { { {B } } } } } { \mathbf {E } ^ { 2 } + \mathbf {B } ^ { 2 } }.
$$
