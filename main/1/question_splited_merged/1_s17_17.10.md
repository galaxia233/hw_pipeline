Problem 17.10. Show that as a test particle approaches $\mathbf{r} = \mathbf{r}_{+}$ , the horizon of a Kerr black hole, it has "angular velocity as seen from infinity" equal to
$$
\Omega \equiv \frac {\mathrm {d} \phi}{\mathrm {d} t} = \frac {\mathrm {a}}{2 \mathrm {M r} _ {+}}.
$$

## Solution

Solution 17.10. In Problem 17.8 we saw that $\Omega \to \mathfrak{a} / 2\mathbf{M}\mathbf{r}_{+}$ when $\theta = \pi /2$ To prove this is so for any $\theta$ we write, in the notation of Problem 17.9:
$$
\Omega = \frac {\mathrm {p} ^ {\phi}}{\mathrm {p} ^ {\mathrm {t}}} = \frac {\mathrm {g} ^ {\phi \phi} \mathrm {p} _ { \phi} + \mathrm {g} ^ { \phi \mathrm {t}} \mathrm {p} _ { \mathrm {t}}}{\mathrm {g} ^ { \mathrm {t t}} \mathrm {p} _ { \mathrm {t}} + \mathrm {g} ^ { \phi \mathrm {t}} \mathrm {p} _ { \phi}} = \omega - \mathrm {e} ^ {2 \nu} \frac { \mathrm {e} ^ { - 2 \psi} \mathrm {p} _ { \phi} } { \mathrm {p} _ { \mathrm {t}} + \omega \mathrm {p} _ { \phi} }.
$$
At $\Delta = 0$ (that is, on the horizon) the metric function $e^{-2\psi}$ is finite but $e^{2\nu}$ vanishes, hence $\Omega = \omega$ and at $\Delta = 0$
$$
\omega = \frac {- g _ { \phi t} } { g _ { \phi \phi} } = \frac { a } { 2 M r _ { +} }.
$$
