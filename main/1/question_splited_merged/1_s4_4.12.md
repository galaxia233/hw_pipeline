Problem 4.12. A particle of charge $\mathbf{q}$ and mass $m$ is coasting through the lab with velocity $v_{\underline{\mathbf{x}}}$ when it encounters a constant $\underline{\mathbf{E}}$ field in the $y$ -direction. Find $y(x)$ , the shape of the particle's subsequent motion.

## Solution

Solution 4.12. The only nonzero components of $\mathbf{F}_{\nu}^{\mu}$ are $\mathbf{F}_{\mathbf{y}}^{\mathbf{t}} = \mathbf{F}_{\mathbf{t}}^{\mathbf{y}} = \mathbf{E}$ . The initial 4-velocity $\mathfrak{u}^a$ of the particle has components $(\gamma, \gamma \mathbf{v}, 0, 0)$ , $\gamma \equiv (1 - \mathbf{v}^2)^{-\frac{1}{2}}$ ; the Lorentz force equations
$$
\mathrm {d} \mathbf {P} ^ { \mu } / \mathrm {d} \tau = \mathrm {qF} _ { \nu } ^ { \mu } \mathbf {u} ^ { \nu }
$$
then give us $\mathrm{d}\mathbf{P}^{\mathbf{x}} / \mathrm{d}\tau = 0$ for $\mu = 1$ and hence
$$
\mathbf {u} ^ { \mathbf {x} } = \gamma \mathbf {v} \quad \tau = \mathbf {x} / \gamma \mathbf {v}.
$$
The equations for $\mu = 0,2$
$$
\mathrm {d} \mathbf {u} ^ { \mathrm {t} } / \mathrm {d} \tau = (\mathrm {q} / \mathrm {m}) \mathbf {E} \mathbf {u} ^ { \mathrm {y} } \quad \mathrm {d} \mathbf {u} ^ { \mathrm {y} } / \mathrm {d} \tau = (\mathrm {q} / \mathrm {m}) \mathbf {E} \mathbf {u} ^ { \mathrm {t} }
$$
can be combined to give the equation
$$
\mathrm {d} ^ { 2 } \mathbf {u} ^ { \mathcal {Y} } / \mathrm {d} \tau ^ { 2 } = (\mathrm {qE} / \mathrm {m}) ^ { 2 } \mathbf {u} ^ { \mathcal {Y} }
$$
with the solution
$$
\mathbf {u} ^ { \mathbf {y} } = \gamma \sinh [ (\mathrm {qE/m}) r ],
$$
where the initial conditions on $\mathbf{u}^{\mathbf{t}}$ and $\mathbf{u}^{\mathbf{y}}$ have been used. Integrating this result with respect to $\tau$ , substituting $\tau = \mathbf{x} / \gamma \mathbf{v}$ , and using the initial condition $\mathbf{y} = 0$ when $\mathbf{x} = 0$ , gives
$$
y = \left(\frac {m}{qE}\right) \gamma \left[ \cosh \left(\frac {qE}{m}\right) \frac {x}{v \gamma} - 1 \right].
$$
