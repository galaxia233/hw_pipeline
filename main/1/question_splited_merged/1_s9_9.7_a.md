Problem 9.7. In a space of fewer than 4 dimensions simple expressions can be given for the Riemann tensor.
(a) What is the Riemann tensor in a 1-dimensional space?

## Solution

# Solution 9.7.
\beta_ {\mathbf {k i j}} = \mathbf {e} _ {\mathbf {k}} \cdot^ {(4)} \nabla_ {\mathbf {i}} \mathbf {e} _ {\mathbf {j}} = ^ {(4)} \Gamma_ {\mathbf {k j i}} = ^ {(3)} \Gamma_ {\mathbf {k j i}}.
$$
The last equality follows from the fact that the Christoffel symbol can be computed directly from ${}^{(3)}\mathbf{g}_{ij}$ . With these coefficients Equation (1) becomes the "Gauss-Weingarten equation."
$$
\left(^ {(4)} \nabla_ {i} e _ {j} = \varepsilon K _ {i j} n + ^ {(3)} \Gamma_ {j i} ^ {k} e _ {k}\right). \tag {2}
$$
The Riemann tensor can be computed from derivatives of the $\Gamma$ 's; it is somewhat quicker to use the curvature operator (see Problem 9.12):
$$
\mathbf {R} _ {\alpha \beta \gamma \delta} = \mathbf {e} _ {\alpha} \cdot \mathbf {R} (\mathbf {e} _ {\gamma}, \mathbf {e} _ {\delta}) \mathbf {e} _ {\beta} \tag {3}
$$
where
$$
\mathbf {R} \left(\mathbf {e} _ {\gamma}, \mathbf {e} _ {\delta}\right) = \left[ \nabla_ {\gamma}, \nabla_ {\delta} \right] - \nabla_ {\left[ \mathbf {e} _ {\gamma}, \mathbf {e} _ {\delta} \right]} . \tag {4}
$$
Evaluate Equation (4) for $\mathbf{e}_{\mathbf{j}}$ and $\mathbf{e}_{\mathbf{k}}$ . The last term vanishes because $[\mathbf{e}_{\mathbf{j}}, \mathbf{e}_{\mathbf{k}}] = 0$ for a coordinate basis, and
$$
\begin{array}{l} { } ^ { ( 4 ) } \nabla _ { j } { } ^ { ( 4 ) } \nabla _ { k } \mathbf { e } _ { i } = { } ^ { ( 4 ) } \nabla _ { j } ( \varepsilon K _ {i k} \mathbf {n} + { } ^ { ( 3 ) } \Gamma_ {i k} ^ {m} \mathbf {e} _ {m} ) \\ = \varepsilon K _ {i k, j} n - \varepsilon K _ {i k} K _ {j} ^ {m} e _ {m} + ^ {(3)} \Gamma_ {i k, j} ^ {m} e _ {m} \\ + ^ {(3)} \Gamma_ {i k} ^ {m} \left(\varepsilon K _ {j m} n + ^ {(3)} \Gamma_ {m j} ^ {n} e _ {n}\right). \tag {5} \\ \end{array}
$$
Evaluate Equation (5) with $j$ and $k$ interchanged and subtract the two equations. The result is
$$
\mathbf {R} \left(\mathbf {e} _ {j}, \mathbf {e} _ {k}\right) \mathbf {e} _ {i} = \varepsilon \left(\mathrm {K} _ {i k} | _ {j} - \mathrm {K} _ {i j} | _ {k}\right) n + \mathbf {e} _ {m} \left[ \varepsilon \left(\mathrm {K} _ {i j} \mathrm {K} _ {k} ^ {\mathrm {m}} - \mathrm {K} _ {i k} \mathrm {K} _ {j} ^ {\mathrm {m}}\right) + ^ {(3)} \mathrm {R} _ {i j k} ^ {\mathrm {m}} \right]. (6)
$$
Dotting Equation (6) with $\mathbf{n}$ and $\mathbf{e}_{\mathbf{n}}$ respectively gives the Gauss-Codazzi equations.
The components of the Riemann tensor with two $\mathbf{n}$ 's can be found from $\mathbf{R}(\mathbf{e}_{\mathbf{k}}, \mathbf{n}) \mathbf{n}$ . Since $\mathbf{n}$ has been chosen to be a coordinate basis vector, $[\mathbf{n}, \mathbf{e}_{\mathbf{k}}] = 0$ so the last term in Equation (4) vanishes and thus
$$
R \left(e _ {k}, n\right) n = ^ {(4)} \nabla_ {k} ^ { \prime } ^ {(4)} \nabla_ {n} n - ^ {(4)} \nabla_ {n} ^ {(4)} \nabla_ {k} n.
$$
But ${}^{(4)}\nabla_{\mathbf{n}}\mathfrak{n} = 0$ because $\mathbf{n}$ is tangent to a geodesic (Gaussian normal coordinates), so that
$$
\mathbf {e} _ {\mathbf {i}} \cdot \mathbf {R} (\mathbf {e} _ {\mathbf {k}}, \mathbf {n}) \mathbf {n} = \mathbf {e} _ {\mathbf {i}} \cdot \mathbf {\Sigma} ^ {(4)} \nabla_ {\mathbf {n}} (\mathbf {K} _ {\mathbf {k}} ^ { \mathrm {m}} \mathbf {e} _ { \mathbf {m}}) = \mathbf { \Sigma } \mathbf {K} _ { \mathbf {k i}, \mathbf {n}} - \mathbf {K} _ { \mathbf {k}} ^ { \mathrm {m}} \mathbf {e} _ { \mathbf {m}} \cdot \nabla_ {\mathbf {n}} \mathbf {e} _ { \mathbf {i}}.
$$
Now $[\mathbf{e},\mathbf{n}] = 0$ implies that
$$
\left(^ {(4)} \nabla_ {\mathbf {n}} \mathbf {e} _ { \mathbf {i}} = ^ {(4)} \nabla_ {\mathbf {i}} \mathbf {n} = - K _ { \mathbf {i}} ^ { \mathbf {j}} \mathbf {e} _ { \mathbf {j}} \right.
$$
so finally
$$
\mathrm {R} _ { \text {i n k n}} = \varepsilon \mathrm {R} _ { \text {i n k}} ^ { \mathrm {n}} = \mathrm {K} _ { \mathrm {k i}, \mathrm {n}} + \mathrm {K} _ { \mathrm {i}} ^ { \mathrm {j}} \mathrm {K} _ { \mathrm {j k}}.
$$
