# Problem 5.17.
(b) Consider an isolated, stressed body at rest and in equilibrium $(\mathbf{T}^{\alpha \beta},_0 = 0)$ in the laboratory frame. Show that its total inertial mass,
defined by
$$
\mathbf {M} ^ {\mathrm {i j}} \equiv \int_ {\substack {\text {stressed} \\ \text {body}}} \mathbf {m} ^ {\mathrm {i j}} \mathrm {d x} \mathrm {d y} \mathrm {d z}
$$
is isotropic and equals the rest mass of the body, i.e. show that
$$
\mathbf {M} ^ {\mathrm {i j}} = \delta^ {\mathrm {i j}} \int \mathbf {T} ^ {0 0} d x d y d z.
$$

## Solution

(b) Since
$$
\mathsf {M} ^ { \mathrm {ij} } \equiv \delta^ { \mathrm {ij} } \int \mathsf {T} ^ {0 0} \mathrm {dx dy dz} + \int \mathsf {T} ^ { \mathrm {ij} } \mathrm {dx dy dz},
$$
we need only show that the second term on the right vanishes. Note that the 3-divergence of $\mathbf{T}^{\mathbf{jk}}$ vanishes
$$
\mathbf {T} _ {, \mathbf {k}} ^ { \mathrm {jk} } = \mathbf {T} _ {, \mu} ^ { \mathrm {j\mu} } - \mathbf {T} _ {, 0} ^ { \mathrm {j0} } = 0 - 0 ,
$$
and that
$$
(\mathbf {x} ^ { \mathbf {k}} \mathbf {T} ^ { \mathbf {ji} }) _ {, \mathbf {i}} = \delta_ { \mathbf {i}} ^ { \mathbf {k}} \mathbf {T} ^ { \mathbf {ji} } + \mathbf {x} ^ { \mathbf {k}} \mathbf {T} _ {, \mathbf {i}} ^ { \mathbf {ji} } = \mathbf {T} ^ { \mathbf {jk} } .
$$
It follows by Gauss' theorem that
$$
\int \mathrm {T} ^ { \mathrm {jk} } \mathrm {dx dydz} = \int \left(\mathrm {x} ^ { \mathrm {k} } \mathrm {T} ^ { \mathrm {ji} }\right) _ {, \mathrm {i}} \mathrm {dxdyd z} = \int_ { \mathrm {S}} \mathrm {x} ^ { \mathrm {k} } \mathrm {T} ^ { \mathrm {ji} } \mathrm {n} ^ { \mathrm {i}} \mathrm {dS}
$$
where $S$ is any surface outside the stressed body and $\underline{n}$ is the outward normal to $S$ . Since the surface integral obviously vanishes, the proof is complete.
