4.20 (b) Problem 4.20.
(b) Show that if $\mathbf{F}$ is a solution of the free-space Maxwell equations, so is $*\mathbf{F}$ and also $\mathbf{e}^{*a}\mathbf{F} \equiv \mathbf{F} \cos \alpha + * \mathbf{F} \sin \alpha$ for arbitrary $a$ . ( $\mathbf{F} \rightarrow \mathbf{e}^{*a}\mathbf{F}$ is called a “duality rotation”).

## Solution

(b) [Note: Since $*(*\mathbf{F}) = -\mathbf{F}$ , if we treat $*$ as an operator it has the same algebraic properties as i, hence we write $e^{*a}\mathbf{F} = \mathbf{F}\cos a + *\mathbf{F}\sin a$ .] According to Problem 4.9 Maxwell's equations, in the absence of sources, are
$$
* \mathbf {F} _ { , \nu } ^ { \mu \nu } = 0 \quad \mathbf {F} _ { , \nu } ^ { \mu \nu } = 0.
$$
Clearly the transformation $\mathbf{F} \rightarrow * \mathbf {F}$ leaves these equations invariant i.e. $*\mathbf{F}$ is a solution if $\mathbf{F}$ is. Since the equations are linear, a linear combination such as $\mathbf{e}^{*\alpha} \mathbf {F}$ must also satisfy the equations.
