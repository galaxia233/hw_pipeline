Problem 4.19. Derive the Lorentz force law for a charged particle from the action $\int \mathrm{J}^{\mu}\mathrm{A}_{\mu}\mathrm{d}^{4}\mathbf{x} - \mathfrak{m}\int \mathrm{d}\tau$ , where $\mathrm{J}^{\mu}$ is the 4-current, $\mathrm{A}_{\mu}$ the vector potential, and $\mathrm{d}\tau^{2} \equiv -\eta_{\alpha \beta}\mathrm{d}\mathbf{x}^{\alpha}\mathrm{d}\mathbf{x}^{\beta}$ .

## Solution

Solution 4.19. From Problem 4.7 we know that, for a particle of charge q
$$
\mathbf {J} ^ { \mu } ( \mathbf {x} ) = \mathrm {q} \int \delta ^ { 4 } \left(\mathbf {x} - \mathbf {z} [ \tau ] \right) \mathbf {u} ^ { \mu } \mathrm {d} \tau ,
$$
so that the action can be written as
$$
\int \mathrm {L} \mathrm {d} r = \mathrm {q} \int \mathrm {u} ^ { \mu } \mathrm {A} _ { \mu } \mathrm {d} r - \mathrm {m} \int \mathrm {d} r
$$
and the Lagrangian is:
$$
\mathbf {L} = \mathbf {q u} ^ { \mu } \mathbf {A} _ { \mu } - \mathbf {m} ( - \eta _ { \alpha \beta } \mathbf {u} ^ { \alpha } \mathbf {u} ^ { \beta } ) ^ { \frac { 1 } { 2 } }.
$$
The coordinates $z^{\lambda}$ specify the position of the particle parametrized by $r$ and
$$
\frac { \partial \mathbf {L} } { \partial ( \mathrm {d} z ^ { \lambda } / \mathrm {d} \tau ) } = \frac { \partial \mathbf {L} } { \partial \mathbf {u} ^ { \lambda } } = \mathbf {qA} _ { \lambda } + \mathbf {mu} _ { \lambda }.
$$
The Euler-Lagrange equations then give us
$$
\frac { \mathrm {d} } { \mathrm {d} \tau } \left(\frac { \partial \mathrm {L} } { \partial \mathrm {u} ^ { \lambda } } \right) = \mathrm {qA} _ { \lambda , \mu } \frac { \mathrm {d} z ^ { \mu } } { \mathrm {d} \tau } + \mathrm {m} \frac { \mathrm {d} \mathrm {u} _ { \lambda } } { \mathrm {d} \tau } = \frac { \partial } { \partial z ^ { \lambda } } \mathrm {L} = \mathrm {qu} ^ { \mu } \mathrm {A} _ { \mu , \lambda },
$$
so that the equations of motion are
$$
\mathrm {md u} _ { \lambda } / \mathrm {d} \tau = \mathrm {q} \left(\mathrm {A} _ { \mu , \lambda } - \mathrm {A} _ { \lambda , \mu } \right) \mathrm {u} ^ { \mu }
$$
or
$$
\mathrm {dp} _ { \lambda } / \mathrm {d} \tau = \mathrm {qF} _ { \lambda \mu } \mathrm {u} ^ { \mu }.
$$
