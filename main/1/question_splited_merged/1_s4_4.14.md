Problem 4.14. A small test particle (mass $m$ , positive charge $q$ ) makes circular orbits around a "fixed" (i.e. very massive) body of positive charge $Q$ . A uniform magnetic field $\underline{B}$ perpendicular to the orbital plane serves to keep the particle in orbit. In the inertial frame in which the central body is at rest, the test charge is seen to circle in the plane perpendicular to the $B$ field with an angular frequency $\omega$ . What is the charge to mass ratio of the test particle in terms of $\omega$ , $R$ , $B$ , $Q$ ?

## Solution

Solution 4.14. From the Lorentz force equation (Problem 4.11),
$$
\frac { \mathrm {d} p } { \mathrm {d} t } = \left(- q \omega R B + \frac { q Q } { R ^ { 2 } } \right) \underset { \sim } { e } _ { r } = m \frac { \mathrm {d} } { \mathrm {d} t } \left(y \underset { \sim } { v } \right) .
$$
No work is being done on the particle so $\gamma = (1 - \omega^{2}\mathbf{R}^{2})^{-\frac{1}{2}}$ is a constant. Since $\mathrm{d}\underline{\mathrm{v}}/\mathrm{d}t = -\omega^{2}\underline{\mathrm{Re}}_{\mathrm{r}}$ we have
$$
\mathrm {q} \omega \mathrm {RB} - \mathrm {qQ} / \mathrm {R} ^ { 2 } = \mathrm {m} \omega ^ { 2 } \mathrm {R} / (1 - \omega^ { 2 } \mathrm {R} ^ { 2 } ) ^ { \frac { 1 } { 2 } }
$$
and hence
$$
\frac { \mathrm {q} } { \mathrm {m} } = \frac { \omega ^ { 2 } } { \left( 1 - \omega^ { 2 } \mathrm {R} ^ { 2 } \right) ^ { \frac { 1 } { 2 } } \left(\omega \mathrm {B} - \mathrm {Q} / \mathrm {R} ^ { 3 } \right) } .
$$
