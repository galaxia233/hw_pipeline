Problem 5.20. Use the equations of motion $(\mathbf{T}^{\mu \nu}; \nu = 0)$ to show that the flow of a perfect fluid is isentropic.

## Solution

Solution 5.20. For a perfect fluid $\mathbf{T}^{\mu \nu} = (\mathbf{P} + \rho) \mathbf{u}^{\mu} \mathbf{u}^{\nu} + \mathbf{P} \mathbf{g}^{\mu \nu}$ so that in the fluid rest frame the 0 component of the equation of motion is
$$
\begin{array}{l} \mathrm {u} _ { \mu } \mathrm {T} ^ { \mu \nu } _ { ; \nu } = \mathrm {T} ^ { 0 \nu } _ { ; \nu } = \mathrm {P} _ { , \nu } \mathrm {g} ^ { 0 \nu } + \mathrm {P g} ^ { 0 \nu } _ { ; \nu } + (\mathrm {P} + \rho) _ { , \nu } \mathrm {u} ^ { \nu } \mathrm {u} ^ { 0 } \\ + (P + \rho) \left[ u _ { ; \nu } ^ { \nu } u ^ { 0 } + u _ { ; \nu } ^ { 0 } u ^ { \nu } \right] = 0 . \\ \end{array}
$$
Now $\mathbf{g}^{\mu \nu}_{;\nu} = 0$ , $\mathbf{u}^{0} = 1$ , and $\mathbf{u}_{; \nu}^{0} = \mathbf{u}_{a} \mathbf{u}_{; \nu}^{a} = \frac{1}{2} (\mathbf{u}_{a} \mathbf{u}_{; \nu}^{a})_{; \nu} = 0$ , so that the equation becomes
$$
0 = - \frac { \mathrm {d} } { \mathrm {d} t } \mathbf {P} + \frac { \mathrm {d} } { \mathrm {d} t } ( \mathbf {P} + \boldsymbol { \rho } ) + ( \mathbf {P} + \boldsymbol { \rho } ) \mathbf {u} _ { ; \nu } ^ { \nu } .
$$
If $n$ is the density of baryons in the fluid rest frame then the number-flux vector of baryons $nu$ is conserved. In the rest frame then
$$
\left( \mathrm {n u} ^ { \nu } \right) _ { ; \nu } = 0 = \mathrm {n} _ { , \nu } \mathrm {u} ^ { \nu } + \mathrm {n u} _ { ; \nu } ^ { \nu } = \frac { \mathrm {d n} } { \mathrm {d t } } + \mathrm {n u} _ { ; \nu } ^ { \nu } .
$$
When this equation is used to eliminate $\mathfrak{u}_{; \nu}^{\nu}$ the equation of motion becomes
$$
\frac { \mathrm {d} \rho } { \mathrm {d} t } = \frac { (P + \rho) } { n } \frac { \mathrm {d} n } { \mathrm {d} t } .
$$
If we compare this with the first law of thermodynamics (see Problem 5.19) we see that $\mathrm{ds} / \mathrm{dt} = 0$ ; the fluid flow is isentropic.
