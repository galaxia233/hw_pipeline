Problem 3.4. Show that the sum of two vectors can be spacelike, null, or timelike, independently of whether the two vectors are spacelike, null, or timelike.

## Solution

Solution 3.4. Take flat Minkowski space, so that $\hat{\mathbf{x}}, \hat{\mathbf{y}}, \hat{\mathbf{z}}$ are spacelike unit vectors; $\hat{\mathbf{t}}$ is a timelike unit vector; $\hat{\mathbf{t}} \pm \hat{\mathbf{z}}, \hat{\mathbf{t}} \pm \hat{\mathbf{y}}$ are null, etc. The following table gives examples in the order: spacelike sum, null sum, timelike sum.
<table><tr><td>spacelike</td><td>null</td><td>timelike</td></tr><tr><td>x̂ + ŷ(x̂ + ε t) + (-x̂ + ε z)(x̂ + ε t) + (-x̂)</td><td>x̂ + (x̂ - t)(x̂ + t) + (-2x̂)(-x̂) + (x̂ - t)</td><td>x̂ + (ε t)x̂ + t(ε x̂) + t</td></tr><tr><td>null</td><td>(x̂ - t) + (x̂ + t)(x̂ + t) + (x̂ + t)(t - x̂) + (t + x̂)</td><td>(x̂ - t) + t(x̂ - t) + 2t(x̂ + t) + t</td></tr><tr><td>timelike</td><td>spacelike sum → null sum → timelike sum →</td><td>(t + ε x̂) + (-t)(t + ε x̂) + (-t + ε t)t + t</td></tr></table>
Here $\varepsilon$ represents any small constant, e.g. 0.1. Not all possibilities would be allowed if one required the timelike vectors to be “future directed.” i.e. $\mathbf{u} \cdot \hat{\mathbf{t}} < 0$ for timelike vectors $\mathbf{u}$ . (The reader can work this out.)
