Problem 4.17. If $\mathbf{T}^{\mu \nu}$ is the stress-energy tensor of the electromagnetic field, show that
$$
\mathbf {T} _ {\alpha} ^ {\mu} \mathbf {T} _ {\nu} ^ {a} = \delta_ {\nu} ^ {\mu} \left[ \left(\mathbf {E} ^ {2} - \mathbf {B} ^ {2}\right) ^ {2} + \left(2 \underline {{\mathbf {E}}} \cdot \underline {{\mathbf {B}}}\right) ^ {2} \right] / (8 \pi) ^ {2}.
$$

## Solution

Solution 4.17. It is sufficient to show that this relation is true in one frame.
$$
\text {C a s e 1 :} \quad \mathrm {E} ^ { 2 } = \mathrm {B} ^ { 2 } , \quad \underline { { { \mathrm {E } } } } \cdot \underline { { { \mathrm {B } } } } = 0.
$$
Choose $\mathbf{E} = \varepsilon \mathbf{e}_{\mathbf{x}}$ , $\mathbf{B} = \varepsilon \mathbf{e}_{\mathbf{y}}$ ; then the only nonvanishing components of $\mathbf{T}^{\mu \nu}$ are $\mathbf{T}^{00} = \mathbf{T}^{0z} = \mathbf{T}^{zz} = \varepsilon^2 / 4\pi$ from which it follows that $\mathbf{T}_{\alpha}^{\mu} \mathbf{T}_{\nu}^{\alpha} = 0$ .
$$
\text {C a s e} 2: \left(E ^ { 2 } - B ^ { 2 } \right) ^ { 2 } + (\underline { { E } } \cdot \underline { { B } } ) ^ { 2 } \neq 0.
$$
From Problem 4.5 we can make $\underline{\mathbf{E}}$ and $\underline{\mathbf{B}}$ parallel. Choose $\underline{\mathbf{E}} = \mathbf{E}\underline{\mathbf{e}}_{\mathbf{x}}$ , $\underline{\mathbf{B}} = \mathbf{B}\underline{\mathbf{e}}_{\mathbf{x}}$ . The nonvanishing components of the stress-energy are then $\mathbf{T}^{00} = -\mathbf{T}^{\mathbf{x}\mathbf{x}} = \mathbf{T}^{\mathbf{y}\mathbf{y}} = \mathbf{T}^{\mathbf{z}\mathbf{z}} = (1 / 8\pi)(\mathbf{E}^{2} + \mathbf{B}^{2})$ and
$$
\begin{array}{l} \mathrm {T} _ { \alpha } ^ { \mu } \mathrm {T} _ { \nu } ^ { \alpha } = \delta _ { \nu } ^ { \mu } \left[ \mathrm {E} ^ { 2 } + \mathrm {B} ^ { 2 } \right] ^ { 2 } / (8 \pi ) ^ { 2 } = \delta _ { \nu } ^ { \mu } \left[ (\mathrm {E} ^ { 2 } - \mathrm {B} ^ { 2 } ) ^ { 2 } + 4 \mathrm {E} ^ { 2 } \mathrm {B} ^ { 2 } \right] / (8 \pi ) ^ { 2 } \\ = \delta _ { \mathcal {V} } ^ { \mu } \left[ \left(\mathrm {E} ^ { 2 } - \mathrm {B} ^ { 2 } \right) ^ { 2 } + \left(2 \underline { { { \mathrm {E } } } } \cdot \underline { { { \mathrm {B } } } } \right) ^ { 2 } \right] / (8 \pi ) ^ { 2 } . \\ \end{array}
$$
