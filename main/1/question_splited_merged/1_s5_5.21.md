Problem 5.21. For a perfect fluid with equation of state $\rho = \rho (\mathfrak{n})$ (where $\mathfrak{n} =$ baryon density) show that $\mathbf{T}_{\mu}^{\mu}$ , the trace of the stress-energy tensor is negative if and only if
$$
d \log \rho / d \log n <   4 / 3.
$$

## Solution

Solution 5.21. From the first law of thermodynamics for a perfect fluid (see Problem 5.20) having equation of state $\rho = \rho (\mathfrak{n})$
$$
\frac { \mathrm {d} \rho } { \rho } = \frac { \rho + \mathbf {P} } { \rho } \frac { \mathrm {d} n } { \mathbf {n} }
$$
so that
$$
d \log \rho / d \ln n = (\rho + P) / \rho .
$$
Thus $\rho$ is greater than 3P if and only if $\mathsf{d}\log \rho /\mathsf{d}\log \mathfrak{n} < 4 / 3$ . Since
$$
\mathrm {T} _ { \mu } ^ { \mu } = ( \rho + \mathrm {P} ) \mathrm {u} ^ { \mu } \mathrm {u} _ { \mu } + \mathrm {p g} _ { \mu } ^ { \mu } = - ( \rho + \mathrm {P} ) + 4 \mathrm {P} = 3 \mathrm {P} - \rho
$$
for a perfect fluid, the proof is complete.
