Problem 4.23. Find the equation for the convective time rate of change of a magnetic field which is "frozen in" a perfectly conducting fluid, in terms of the expansion, shear, and rotation of the fluid. (See Problem 5.18 for definitions of these quantities.)

## Solution

Solution 4.23. Since the conductivity is infinite, the Lorentz force on a charge carrier must be zero, so $\underline{\mathbf{E}} +\underline{\mathbf{v}}\times \underline{\mathbf{B}} = 0$ . Maxwell's equation $\dot{\underline{\mathbf{B}}} = -\underline{\nabla}\times \underline{\mathbf{E}}$ gives
$$
\underline { { { \mathbf {B } } } } = \underline { { { \nabla } } } \times ( \underline { { { \mathbf {v } } } } \times \underline { { { \mathbf {B } } } } ) = - \underline { { { \mathbf {B } } } } ( \underline { { { \nabla } } } \cdot \underline { { { \mathbf {v } } } } ) + ( \underline { { { \mathbf {B } } } } \cdot \underline { { { \nabla } } } ) \underline { { { \mathbf {v } } } } - ( \underline { { { \mathbf {v } } } } \cdot \underline { { { \nabla } } } ) \underline { { { \mathbf {B } } } } . \tag {1}
$$
If we now specialize to the instantaneous rest frame of the fluid, $\underline{\mathbf{v}} = 0$ , $t = \tau$ (proper time), then $\gamma = 1$ and has vanishing first derivatives, and $\partial/\partial t$ becomes $\partial/\partial \tau$ (i.e. the convective derivative and the partial derivative are equal), so
$$
d B ^ { i } / d \tau = - B ^ { i } v ^ { j } , _ { j } + B ^ { j } v ^ { i } , _ { j } = - B ^ { i } u ^ { j } , _ { j } + B ^ { j } u ^ { i } , _ { j }.
$$
Since $\mathbf{u}_{,j}^{i}$ is the projection perpendicular to $\mathbf{u}$ , of $\mathbf{u}_{,j}^{\mu}$ it follows that
$$
u _ { i , j } = \omega _ { i j } + \sigma _ { i j } + \frac {1}{3} g _ { i j } \theta .
$$
The contraction of this 3-tensor is of course $\mathbf{u}_{,i}^{i} = \theta$ and the proper time derivative of $\mathbf{B}$ is then
$$
\mathrm {d} \mathbf {B} ^ { \mathrm {i} } / \mathrm {d} \tau = - \mathbf {B} ^ { \mathrm {i} } \theta + \mathbf {B} _ { \mathrm {j} } \left(\omega ^ { \mathrm {i j} } + \sigma ^ { \mathrm {i j} } + \frac {1}{3} \mathbf {g} ^ { \mathrm {i j} } \theta \right) = - \frac { 2 } { 3 } \mathbf {B} ^ { \mathrm {i} } \theta + \left(\sigma ^ { \mathrm {i j} } + \omega ^ { \mathrm {i j} } \right) \mathbf {B} _ { \mathrm {j} } . \tag {2}
$$
To Equation (2) may be added any term proportional to the spatial component of the 4-velocity (since that vanishes in the comoving frame). Thus write Equation (2) as
$$
\mathrm {d} \mathbf {B} ^ { \mathrm {i} } / \mathrm {d} \tau = - \frac { 2 } { 3 } \mathbf {B} ^ { \mathrm {i} } \theta + \left(\sigma ^ { \mathrm {i j} } + \omega ^ { \mathrm {i j} } \right) \mathbf {B} _ { \mathrm {j} } + \mathbf {fu} ^ { \mathrm {i} } \tag {3}
$$
where $f$ is as yet arbitrary. Now transform Equation (2) into a tensor equation by
(a) Letting indices run from 0-3 instead of just 1-3;
(b) Defining a 4-vector $\mathbf{B}^{\alpha}$ by $\mathbf{B}^{\alpha}\mathbf{u}_{\alpha} = 0$ and the spatial part of $\mathbf{B}$ equal in the comoving frame to $\widetilde{\mathbf{B}}$ ;
(c) Changing ordinary derivatives to covariant derivatives;
(d) Defining $f$ so that $d(\mathbf{B} \cdot \mathbf{u}) / d\tau = 0$ , i.e. so that the normalization of $\mathbf{B}$ is preserved.
The unique equation satisfying (a), (b), (c), (d) is
$$
D B ^ { \alpha } / d \tau = u ^ { \alpha } a _ { \beta } B ^ { \beta } + \left(\omega ^ { \alpha \beta } + \sigma ^ { \alpha \beta } \right) B _ { \beta } - \frac { 2 } { 3 } \theta P ^ { \alpha } \beta^ { B ^ { \beta } } \tag {4}
$$
where $\mathbf{a}$ is the 4-acceleration, and $\mathbf{P}^{\alpha}_{\beta}$ is the projection operator defined in Problem 5.18. Equation (4) is obviously a tensor equation and reduces to Equations (3) and (1) in a comoving orthonormal frame, thus it is the correct equation in all frames.
