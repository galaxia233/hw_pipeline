Problem 5.32. As in non-relativistic thermodynamics, one defines the specific heat of a gas at constant volume and constant pressure by
$$
\mathbf {c _ {v}} = \mathrm {T} \left. \frac {\mathrm {d s}}{\mathrm {d T}} \right| _ {\mathbf {n}} \qquad \mathbf {c _ {p}} = \mathrm {T} \left. \frac {\mathrm {d s}}{\mathrm {d T}} \right| _ {\mathbf {p}} .
$$
For a perfect Maxwell-Boltzmann gas, show that $c_{\mathbf{p}} = c_{\mathbf{v}} + \mathbf{k}$ . (Here $\mathbf{k} =$ Boltzmann's constant.) Show that the adiabatic index
$$
\Gamma_ {1} \equiv \frac {\partial \log p}{\partial \log n} | _ {s}
$$
is equal to the ratio of specific heats,
$$
\gamma \equiv c _ {\mathbf {p}} / c _ {\mathbf {v}}.
$$

## Solution

Solution 5.32. For a perfect Maxwell-Boltzmann gas,
$$
p = n k T \tag { 1 }
$$
$$
\rho / \mathfrak {n} = \mathrm {U} ( \mathrm {T } ) \tag { 2 }
$$
Equation (2) says that the energy per particle is a function of temperature alone. From the first law of relativistic thermodynamics,
$$
\begin{array}{l} \mathrm {T d s } = \mathrm {d} ( \rho / \mathrm {n } ) + \mathrm {p d } ( 1 / \mathrm {n } ) \\ = \frac { \mathrm {d} U } { \mathrm {d} T } \mathrm {d} T + \mathrm {p d } \left( \frac { 1 } { n } \right) \tag { 3 } \\ \end{array}
$$
we see that
$$
c _ {v } = \frac { d U } { d T } . \tag { 4 }
$$
Equation (1) implies that
$$
\mathrm {p d } \left( \frac { 1 } { \mathrm {n } } \right) + \frac { 1 } { \mathrm {n } } \mathrm {d p } = \mathrm {k dT } \tag { 5 }
$$
so Equation (3) becomes
$$
\mathrm {T d s } = \left( \frac { \mathrm {d} U } { \mathrm {d} T } + \mathrm {k } \right) \mathrm {d} T - \frac { 1 } { \mathrm {n } } \mathrm {d p } \tag { 6 }
$$
and thus
$$
c _ {p } = \frac { d U } { d T } + k = c _ {v } + k . \tag { 7 }
$$
From Equation (1)
$$
\Gamma_ { 1 } = \left. \frac { \partial \log p } { \partial \log n } \right| _ {s} = 1 + \frac { n } { T } \frac { d T } { d n } \Bigg | _ {s} \tag { 8 }
$$
But Equation (3) implies that, when $\mathbf {s} =$ constant,
$$
c _ {v } \frac { d T } { d n } = \frac { p } { n ^ { 2 } } = k \frac { T } { n } = \left( c _ {p } - c _ {v } \right) \frac { T } { n } . \tag { 9 }
$$
Substituting Equation (9) in Equation (8) gives us
$$
\Gamma_ { 1 } = 1 + \frac { \left( \mathrm {c} _ { \mathrm {p } } - \mathrm {c} _ { \mathrm {v } } \right) } { \mathrm {c} _ { \mathrm {v } } } = \gamma .
$$
