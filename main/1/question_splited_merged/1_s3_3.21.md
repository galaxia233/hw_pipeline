Problem 3.21. In an orthonormal frame, show that
$$
\varepsilon_ {\alpha \beta \gamma \delta} = - \varepsilon^ {\alpha \beta \gamma \delta}.
$$
What is the analogous relation in a general coordinate frame with metric $\mathbf{g}_{\mu \nu}$ ?

## Solution

Solution 3.21. In an orthonormal frame
$$
\varepsilon^ { \alpha \beta \gamma \delta } = \eta^ { \alpha \mu } \eta^ { \beta \nu } \eta^ { \gamma \sigma } \eta^ { \delta \lambda } \varepsilon_ { \mu \nu \sigma \lambda }.
$$
For nonzero components of $\varepsilon^ { \alpha \beta \gamma \delta }$ , precisely one of the indices $a, \beta, \gamma, \delta$ must equal zero. Since $\eta^{00} = -1$ and $\eta^{jj} = 1$ this introduces precisely one minus sign in what is otherwise an identity transformation. Thus
$$
\varepsilon_ { \alpha \beta \gamma \delta } = - \varepsilon^ { \alpha \beta \gamma \delta }.
$$
In some other frame, this equation becomes
$$
\left[ - \det  \left(\mathbf {g} _ { \mu \nu }\right) \right] ^ { - \frac {1}{2} } \varepsilon_ { \alpha } \beta \gamma \delta = - \left[ - \det  \left(\mathbf {g} _ { \mu \nu }\right) \right] ^ { \frac {1}{2} } \varepsilon^ { \alpha \beta \gamma \delta }
$$
according to the transformation derived in Solution 3.20, and its obvious analog for $\varepsilon^{\alpha \beta \gamma \delta}$ . Thus in general
$$
\varepsilon_ { a \beta \gamma \delta } = \det  (\mathbf {g} _ { \mu \nu }) \varepsilon^ { a \beta \gamma \delta }.
$$
