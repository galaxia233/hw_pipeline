Problem 4.4. Show that $\mathfrak{S}^2 - |\underline{\mathbf{S}}|^2$ is a Lorentz invariant of the electromagnetic field, where $\mathfrak{S}$ is the energy density and $\underline{\mathbf{S}}$ the Poynting flux.

## Solution

Solution 4.4. The expression is shown to be invariant by relating it to the invariants $\mathbf{E}^2 -\mathbf{B}^2$ and $\underline{\mathbf{E}}\cdot \underline{\mathbf{B}}$
$$
\begin{array}{l} 6 4 \pi (\mathfrak {S } ^ { 2 } - | \underline { { { \mathbf {S } } } } | ^ { 2 } ) = ( \mathbf {E } ^ { 2 } + \mathbf {B } ^ { 2 } ) ^ { 2 } - 4 ( \underline { { { \mathbf {E } } } } \times \underline { { { \mathbf {B } } } } ) ^ { 2 } \\ = \left[ \mathbf {E } ^ { 4 } + 2 \mathbf {E } ^ { 2 } \mathbf {B } ^ { 2 } + \mathbf {B } ^ { 4 } \right] - 4 \left[ \mathbf {E } ^ { 2 } \mathbf {B } ^ { 2 } - ( \underline { { { \mathbf {E } } } } \cdot \underline { { { \mathbf {B } } } } ) ^ { 2 } \right] \\ = \mathrm {E } ^ { 4 } - 2 \mathrm {E } ^ { 2 } \mathrm {B } ^ { 2 } + \mathrm {B } ^ { 4 } + 4 ( \underline { { \mathrm {E } } } \cdot \underline { { \mathrm {B } } } ) ^ { 2 } \\ = \left(\mathrm {E } ^ { 2 } - \mathrm {B } ^ { 2 } \right) ^ { 2 } + 4 ( \underset { \sim } { \mathrm {E } } \cdot \underset { \sim } { \mathrm {B } } ) ^ { 2 } . \\ \end{array}
$$
