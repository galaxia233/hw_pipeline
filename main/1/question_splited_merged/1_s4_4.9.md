Problem 4.9. If $\mathbf{F}^{\mu \nu}$ is the electromagnetic tensor, show that Maxwell's equations in vacuum can be written as $\mathbf{F}_{,\nu}^{\mu \nu} = 0$ and ${\bf *F}^{\mu \nu}_{,\nu} = 0$ . [Here, ${\bf *F}^{\mu \nu}$ is the dual of $\mathbf{F}^{\mu \nu}$ ; see Problem 3.25.]

## Solution

Solution 4.9. It has already been shown (Problem 4.8) that $\mathbf{F}^{\mu \nu}_{,\nu} = 0$ is equivalent to the "electric" Maxwell equations (in vacuum). As for the magnetic equations:
$$
\begin{array}{l} * \mathrm {F} ^ { \mu \nu }, \nu = \frac {1}{2} \left(\mathrm {F} _ { \alpha \beta } \varepsilon ^ { \alpha \beta \mu \nu }\right) , \nu = \frac {1}{2} \mathrm {F} _ { \alpha \beta } , \nu \varepsilon ^ { \alpha \beta \mu \nu } \\ = \frac {1}{2} F _ { [ \alpha \beta , \nu ]} \varepsilon ^ { \alpha \beta \mu \nu }, \\ \end{array}
$$
so that ${\bf \nabla}^{*}{\bf F}^{\mu \nu}_{,\nu} = 0$ is equivalent to $\mathbf{F}_{a\beta ,\nu} + \mathbf{F}_{\nu a,\beta} + \mathbf{F}_{\beta \nu ,a} = 0.$
