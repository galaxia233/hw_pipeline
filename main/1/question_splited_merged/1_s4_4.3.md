Problem 4.3. A particular electromagnetic field has its E field at an angle $\theta_0$ to its B field, and $\theta_0$ is invariant to all observers. What is the value of $\theta_0$ ?

## Solution

Solution 4.3. In any frame the angle is given by
$$
\cos \theta _ { 0 } = \underline { { { \mathbf {E } } } } \cdot \underline { { { \mathbf {B } } } } / | \mathbf {E } | | \mathbf {B } |.
$$
Now $\underline{\mathbf{E}}\cdot \underline{\mathbf{B}}$ is invariant (see Problem 4.2), but $|\mathbf{E}||\mathbf{B}|$ is not. Thus, $\theta_0$ can be invariant if and only if $\underline{\mathbf{E}}\cdot \underline{\mathbf{B}} = 0$ , i.e. $\theta_0 = \pi /2$ .
