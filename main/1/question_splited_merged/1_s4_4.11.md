Problem 4.11. From the spatial components of the Lorentz 4-force equation, find an equation for $\underline{\mathrm{dP}}/\mathrm{dt}$ in terms of $\underline{\mathbf{E}}$ and $\underline{\mathbf{B}}$ . (Here $\underline{\mathbf{P}}$ is the spatial part of $\mathbf{P}$ ).

## Solution

Solution 4.11. The spatial components of the Lorentz equation are
$$
\frac { \mathrm {d} \mathbf {P} ^ { \mathbf {i} } } { \mathrm {d} \tau } = \gamma \frac { \mathrm {d} \mathbf {P} ^ { \mathbf {i} } } { \mathrm {d} t } \mathbf {u} _ { \mu } = q \mathbf {F} ^ { \mathbf {i} \mu } \mathbf {u} _ { 0 } + q \mathbf {F} ^ { \mathbf {i j } } \mathbf {u} _ { \mathbf {j} } = q \gamma \mathbf {E} ^ { \mathbf {i} } + q \gamma \varepsilon ^ { \mathbf {i j k } } \mathbf {B} _ { \mathbf {k} } \mathbf {v} _ { \mathbf {j} },
$$
so that
$$
\mathrm {d} \underline { { { \mathbf {P } } } } / \mathrm {d} t = \mathrm {q} ( \underline { { { \mathbf {E } } } } + \underline { { { \mathbf {v } } } } \times \underline { { { \mathbf {B } } } }).
$$
