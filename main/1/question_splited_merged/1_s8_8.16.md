Problem 8.16. Show that
$$
\mathcal {L} _ {\bf u} \mathcal {L} _ {\bf v} - \mathcal {L} _ {\bf v} \mathcal {L} _ {\bf u} = \mathcal {L} _ {[ \bf u, v ]}.
$$

## Solution

Solution 8.16. If the operator identity holds for scalar functions and for vector fields, it will hold for arbitrary tensors. This is because the action of the Lie derivative on arbitrary tensors is determined by its action on scalars and vectors (see Problem 8.13). Let $f$ be a scalar function, then
$$
\mathcal {L} _ {\bf u} \mathcal {L} _ {\bf v} f - \mathcal {L} _ {\bf v} \mathcal {L} _ {\bf u} f = \mathcal {L} _ {\bf u} \nabla_ {\bf v} f - \mathcal {L} _ {\bf v} \nabla_ {\bf u} f = [ {\bf u}, {\bf v} ] f = \mathcal {L} _ { [ { \bf u } , { \bf v } ] } f.
$$
Now let $w$ be a vector field, then
$$
\mathcal {L} _ {\mathbf {u}} \mathcal {L} _ {\mathbf {v}} w - \mathcal {L} _ {\mathbf {v}} \mathcal {L} _ {\mathbf {u},} w - \mathcal {L} _ { [ \mathbf {u } , \mathbf {v } ] } w = [ \mathbf {u} , [ \mathbf {v} , w ] ] - [ \mathbf {v} , [ \mathbf {u} , w ] ] - [ [ \mathbf {u } , \mathbf {v } ] , w ] = 0 .
$$
The vanishing of this last expression (the Jacobi identity for commutators) can easily be checked e.g. by writing out all terms. This completes the proof for scalars and for vectors, and hence in general.
