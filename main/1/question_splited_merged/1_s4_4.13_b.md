4.13 (b) Problem 4.13. A particle of charge $q$ , mass $m$ , moves in a circular orbit of radius $R$ in a uniform B field $\mathbf{B}_{\underline{\mathbf{e}}_z}$ . (b) The speed of the particle is constant since the B field can do no work on the particle. An observer moving at velocity $\beta_{\underline{\mathbf{e}}_{\mathbf{x}}}$ , however, does not see the speed as constant. What is $\mathfrak{u}^{0^{'}}$ measured by this observer?

## Solution

(b) The 4-velocity components seen in the lab are
$$
u ^ { 0 } = \left(1 - \omega^ { 2 } R ^ { 2 }\right) ^ { - \frac { 1 } { 2 } } . \quad u ^ { x } = \omega y \left(1 - \omega^ { 2 } R ^ { 2 }\right) ^ { - \frac { 1 } { 2 } } \quad u ^ { y } = - \omega x \left(1 - \omega^ { 2 } R ^ { 2 }\right) ^ { - \frac { 1 } { 2 } } .
$$
The components measured by the moving observer are found with a Lorentz transformation, e.g.
$$
u ^ { 0 ^ { \prime } } = \gamma (u ^ { 0 } - \beta u ^ { x } ) = \gamma (1 - \beta \omega y ) \left(1 - \omega^ { 2 } R ^ { 2 }\right) ^ { - \frac { 1 } { 2 } } ,
$$
where $\gamma \equiv (1 - \beta^2)^{-\frac{1}{2}}$
