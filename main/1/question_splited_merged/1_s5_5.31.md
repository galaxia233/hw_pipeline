Problem 5.31. From the stress-energy
$$
\mathbf {T} ^ {\alpha \beta} = \rho \mathbf {u} ^ {\alpha} \mathbf {u} ^ {\beta} + \mathrm {p P} ^ {\alpha \beta} - 2 \eta \sigma^ {\alpha \beta} - \zeta \theta \mathbf {P} ^ {\alpha \beta},
$$
show that the equations of motion derived from $\mathbf{T}^{\alpha \beta},\beta = 0$ reduce to the Navier-Stokes equations in the nonrelativistic limit.

## Solution

Solution 5.31. We know that projecting $\mathbf{T}^{\alpha \beta}_{,\beta}$ along $\mathfrak{u}_\alpha$ gives an equation of local energy conservation; to get an equation of motion for the fluid, we project perpendicular to $\mathfrak{u}_\alpha$ .
$$
\begin{array}{l} 0 = \mathrm {P} _ { \alpha } ^ { \gamma } \mathrm {T} ^ { \alpha \beta } , \beta = \mathrm {P} _ { \alpha } ^ { \gamma } ( \rho , \beta \mathrm {u} ^ { \alpha } \mathrm {u} ^ { \beta } + \rho \mathrm {u} ^ { \alpha } , \beta \mathrm {u} ^ { \beta } + \rho \mathrm {u} ^ { \alpha } \mathrm {u} ^ { \beta } , \beta + \mathrm {p } , \beta \mathrm {P} ^ { \alpha \beta } \\ + \mathrm {p P} ^ { \alpha \beta } , \beta - 2 \eta , \beta^ { \sigma ^ { \alpha \beta } } - 2 \eta^ { \sigma ^ { \alpha \beta } } , \beta - \zeta , \beta^ { \theta \mathrm {P} ^ { \alpha \beta } } - \zeta \theta , \beta^ { \mathrm {P} ^ { \alpha \beta } } - \zeta \theta \mathrm {P} ^ { \alpha \beta } , \beta ) . \\ \end{array}
$$
We now use the following identities: $\mathbf {P}_{\alpha}^{\gamma}\mathfrak{u}^{\alpha} = 0$
$$
\begin{array}{l} \mathrm {P} _ { \alpha } ^ { \gamma } \mathrm {u} _ { , \beta } ^ { \alpha } = \mathrm {u} _ { , \beta } ^ { \gamma } + \mathrm {u} _ { \alpha } ^ { \gamma } \mathrm {u} _ { , \beta } ^ { \alpha } = \mathrm {u} _ { , \beta } ^ { \gamma } \\ \mathbf {P} _ { a } ^ { \gamma } \mathbf {P} ^ { a \beta } = \mathbf {P} ^ { \gamma \beta } \\ \end{array}
$$
$$
\begin{array}{l} \mathrm {P} _ { \alpha } ^ { \gamma } \mathrm {P} _ { , \beta } ^ { \alpha \beta } = \mathrm {P} _ { \alpha } ^ { \gamma } ( \mathrm {u} ^ { \alpha } \mathrm {u} ^ { \beta } ) , \beta = \mathrm {u} ^ { \gamma } , \beta \mathrm {u} ^ { \beta } \\ \mathrm {P} _ { \alpha } ^ { \gamma } \sigma ^ { \alpha \beta } = \sigma ^ { \gamma \beta } \\ \end{array}
$$
$$
\mathbf {P} _ { \alpha } ^ { \gamma } \sigma ^ { \alpha \beta } , \beta = \sigma ^ { \gamma \beta } , \beta + \mathfrak {u} ^ { \gamma } \mathfrak {u} _ { \alpha } \sigma ^ { \alpha \beta } , \beta = \sigma ^ { \gamma \beta } , \beta - \mathfrak {u} ^ { \gamma } \mathfrak {u} _ { \alpha } , \beta \sigma ^ { \alpha \beta } = \sigma ^ { \gamma \beta } , \beta - \mathfrak {u} ^ { \gamma } \sigma_ { \alpha } \beta \sigma ^ { \alpha \beta } .
$$
With these identities the projected equations of motion become:
$$
\begin{array}{l} 0 = ( \rho + p ) u ^ { \gamma } , \beta u ^ { \beta } + p , \beta P ^ { \alpha \beta } - 2 ( \eta \sigma^ { \gamma \beta } + \zeta \theta P ^ { \gamma \beta } ) , \beta \\ + 2 \eta \mathrm {u} ^ { \gamma } \sigma_ { \alpha \beta } \sigma ^ { \alpha \beta } + \zeta \theta^ { 2 } \mathrm {u} ^ { \gamma } . \tag { 1 } \\ \end{array}
$$
In the nonrelativistic limit,
$$
u ^ { t } \approx 1, \quad u ^ { j } \approx v ^ { j } , \quad p = \mathcal {O} (v ^ { 2 } ) , \quad \rho = \mathcal {O} ( 1 ) .
$$
Taking the j-component of Equation (1), and working to $\mathcal{O}(\mathbf{v}^2)$ , we get
$$
\begin{array}{l} 0 = \rho ( \mathrm {v} _ { , t } ^ { \mathrm {j } } + \mathrm {v} _ { , k } ^ { \mathrm {j } } \mathrm {v} ^ { \mathrm {k } } ) + \mathrm {p} _ { , j } \\ - \left[ \eta \left( \mathrm {v} _ { \mathrm {j } , \mathrm {k } } + \mathrm {v} _ { \mathrm {k } , \mathrm {j } } - \frac { 2 } { 3 } \delta_ { \mathrm {j k } } \mathrm {v} ^ { \mathrm {m } } , \mathrm {m} \right) \right] , _ { \mathrm {k } } + \left( \zeta \mathrm {v} ^ { \mathrm {m } } , \mathrm {m} \right) , _ { \mathrm {j } } \\ \end{array}
$$
which is the Navier-Stokes equation.
