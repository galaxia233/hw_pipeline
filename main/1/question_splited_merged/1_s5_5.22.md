Problem 5.22. Show that the velocity of sound $\mathbf{v}_{\mathbf{s}}$ in a relativistic perfect fluid is given by
$$
v _ {s} ^ {2} = \left. \partial p / \partial \rho \right| _ {s = c o n s t a n t}.
$$
For a high temperature relativistic gas with an equation of state $\rho \approx 3\mathrm{P}$ (essentially that for a photon gas) show that $\mathbf{v}_{\mathrm{s}} \approx 1 / \sqrt{3}$ .

## Solution

Solution 5.22. Assume that the acoustical wave is a perturbation (isentropic) in a uniform static fluid with parameters $\rho_0, \mathbf{P}_0$ , and $\mathfrak{n}_0$ . Let the perturbations be $\rho_1, \mathbf{P}_1$ , and $\mathfrak{n}_1$ and the fluid velocity be $\mathbf{u} \approx (1, \underline{\nu}_1)$ in the rest frame of the unperturbed fluid. The first order perturbation terms in $\mathrm{T}_{,\nu}^{\mu \nu} = 0$ give us
(i) From $\mu = 0$ $\nabla \cdot \underline{\mathbf{v}}_1 = -\frac{\partial\rho_1}{\partial t} (\rho_0 + \mathbf{P}_0)^{-1}$   
(ii) From $\mu = 1,2,3$
which can be combined into
$$
\partial ^ { 2 } \rho _ { 1 } / \partial t ^ { 2 } - \nabla ^ { 2 } \mathbf {P _ { 1 } } = 0 .
$$
Since $\mathbf{P}_1$ and $\rho_1$ are related for isentropic flow by
$$
\mathbf {P _ { \lambda _ { 1 } } } = \left. \partial \mathbf {P} / \partial \rho \right| _ { \mathbf {s } } \rho _ { \lambda _ { 1 } } ,
$$
the above equation becomes a wave equation
$$
\nabla ^ { 2 } \rho _ { 1 } - \frac { 1 } { v _ { \mathrm {s } } ^ { 2 } } \frac { \partial ^ { 2 } \rho _ { 1 } } { \partial t ^ { 2 } } = 0
$$
with characteristic velocity $\mathbf{v}_{\mathrm{s}}$
For an equation of state $\rho \approx 3\mathbb{P}$
$$
v _ { s } ^ { 2 } = \partial P / \partial \rho = 1 / 3
$$
so that $\mathbf{v_s} \approx 3^{-\frac{1}{2}}$ in a highly relativistic gas.
