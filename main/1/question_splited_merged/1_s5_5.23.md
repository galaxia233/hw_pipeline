Problem 5.23. The velocity of sound in a fluid is $\left.\mathbf{v}_{\mathbf{s}}^{2} = \frac{\partial \mathbf{p}}{\partial \rho}\right|_{\mathbf{s}} = \text{constant}$ . Show that $\mathbf{v}_{\mathbf{s}}^{2} = \Gamma_{1} \mathbf{p} / (\rho + \mathbf{p})$ where $\Gamma_{1}$ is the adiabatic index
$$
\Gamma_ {1} = \partial \log p / \partial \log n | _ {s = c o n s t a n t}.
$$

## Solution

Solution 5.23. From the first law of thermodynamics (see Problem 5.19)
with $\mathbf{ds} = 0$ we have
$$
\mathrm {d} \mathbf {n} / \mathrm {d} \rho = \mathbf {n} / ( \rho + \mathbf {P } ) .
$$
Now we use the definition of $\Gamma_{1}$ and the expression for $\mathbf{v}_{\mathbf{s}}^{2}$ from Problem 5.21:
$$
\frac { { \bf v } _ { \mathrm {s } } ^ { 2 } } { \Gamma _ { 1 } } = \frac { \mathrm {d P / d } \rho } { ( \mathrm {n d P } ) / ( \mathrm {P d n } ) } = \frac { \mathrm { \bf P } } { \mathrm { \bf n } } \frac { \mathrm {d n } } { \mathrm {d} \rho } = \frac { \mathrm { \bf P } } { \rho + \mathrm { \bf P } } .
$$
