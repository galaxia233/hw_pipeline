Problem 5.24. What is the speed of sound in an ideal Fermi gas at zero temperature?

## Solution

Solution 5.24. At zero temperature all energy states in the Fermi gas are filled up to the Fermi level $\mathbf{E} = \mathbf{E}_{\mathbf{F}}$ . Since there are two spin states for fermions the density of fermions in phase space is $2 / h^3$ so that
$$
\mathrm {d} n / \mathrm {V} = ( 2 / \mathrm {h} ^ { 3 } ) \mathrm {d} ^ { 3 } \mathrm {p}
$$
$$
\rho = \int _ { 0 } ^ { \mathrm {P } _ { \mathrm {F } } } \left( \mathrm {p} ^ { 2 } + \mathrm {m} ^ { 2 } \right) ^ { \frac { 1 } { 2 } } \left( 2 / \mathrm {h} ^ { 3 } \right) 4 \pi \mathrm {p} ^ { 2 } \mathrm {d p}
$$
$$
P = \int _ { 0 } ^ { P _ { F } } \frac { 1 } { 3 } p ^ { 2 } \left( p ^ { 2 } + m ^ { 2 } \right) ^ { - \frac { 1 } { 2 } } ( 2 / h ^ { 3 } ) 4 \pi p ^ { 2 } d p
$$
where $\mathbf{P}_{\mathbf{F}}$ is the momentum corresponding to the Fermi energy. In Problem 5.22 we found $\mathbf{v}_{\mathbf{s}}^{2} = \mathrm{d}\mathbf{P} / \mathrm{d}\rho$ so
$$
\mathrm {v _ {s } ^ { 2 } } = \frac { \mathrm {d P } } { \mathrm {d} \rho } = \frac { \mathrm {d P / d P _ {F } } } { \mathrm {d } \rho / \mathrm {d P _ {F } } } = \frac { 1 } { 3 } \left( \frac { \mathrm {P _ {F } } } { \mathrm {E _ {F } } } \right) ^ { 2 } = \frac { 1 } { 3 } \left( 1 - \frac { \mathrm {m} ^ { 2 } } { \mathrm {E _ {F } } ^ { 2 } } \right) .
$$
Note that for a highly relativistic Fermi gas $\mathbf{v}_{\mathrm{s}} \rightarrow 3^{-\frac{1}{2}}$ .
