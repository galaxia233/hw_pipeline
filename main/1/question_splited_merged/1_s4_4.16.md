Problem 4.16. Show that the stress-energy tensor for the electromagnetic field has zero trace.

## Solution

Solution 4.16. From the stress-energy $\mathbf{T}^{\mu \nu} = (1 / 4\pi)(\mathbf{F}^{\mu a}\mathbf{F}^{\nu}_{a} - \frac{1}{4}\eta^{\mu \nu}\mathbf{F}_{a}\beta \mathbf{F}^{\alpha \beta})$ the trace is easily computed to be
$$
\mathbf {T} _ { \mu } ^ { \mu } = (1 / 4 \pi ) ( \mathbf {F} ^ { \mu \alpha } \mathbf {F} _ { \mu \alpha } - \frac {1}{4} \cdot 4 \cdot \mathbf {F} _ { \alpha \beta } \mathbf {F} ^ { \alpha \beta } ) = 0 .
$$
