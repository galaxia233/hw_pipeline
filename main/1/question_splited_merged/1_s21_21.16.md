Problem 21.16. Let $\mathbf{T_{ab}} = \mathbf{T_{AA'B'B'}}$ in a spinor representation. What tensor corresponds to $\mathbf{T_{BA'AB'}}$ ?

## Solution

Solution 21.16. Since primed and unprimed indices commute, we have
$$
\begin{array}{l} \mathbf {T} _ { \mathrm {(a b)}} = \frac {1}{2} \left(\mathbf {T} _ { \mathbf {A B A ^ { \prime} B ^ { \prime}}} + \mathbf {T} _ { \mathbf {B A B ^ { \prime} A ^ { \prime}}} \right) \\ = \frac {1}{4} \left(\mathrm {T _ {A B A ^ {'} B ^ {'}} - T _ {B A A ^ {'} B ^ {'}} + T _ {B A B ^ {'} A ^ {'}} - T _ {A B B ^ {'} A ^ {'}}} \right) \\ + \frac {1}{2} \mathrm {\Delta T _ {(a b)}} + \frac {1}{4} \left(\mathrm {\Delta T _ {B A A ^ {'} B ^ {'}}} + \mathrm {\Delta T _ {A B B ^ {'} A ^ {'}}} \right) . \\ \end{array}
$$
With the result of Problem 21.14, this can be written as
$$
\begin{array}{l} \mathrm {T} _ { \left(\mathrm {a b}\right)} = 2 \mathrm {T} _ { \left[ \mathrm {A B} \right] \left[ \mathrm {A} ^ { \prime} \mathrm {B} ^ { \prime} \right]} + \frac {1}{2} \left(\mathrm {T} _ { \mathrm {B A A} ^ { \prime} \mathrm {B} ^ { \prime}} + \mathrm {T} _ { \mathrm {A B B} ^ { \prime} \mathrm {A} ^ { \prime}} \right) \\ = \frac {1}{2} \varepsilon_ { \mathbf {A B}} \varepsilon_ { \mathbf {A ^ { \prime} B ^ { \prime}}} \mathrm {T _ {C}} _ { \mathbf {C ^ { \prime}}} ^ { \mathbf {C ^ { \prime}}} + \frac {1}{2} \left(\mathrm {T _ {B A A ^ { \prime} B ^ { \prime}}} + \mathrm {T _ {A B B ^ { \prime} A ^ { \prime}}} \right). \\ \end{array}
$$
Finally, using the result of Problem 21.15, we get
$$
\begin{array}{l} \mathrm {T} _ { \mathrm {B A A ^ { \prime} B ^ { \prime}}} = \mathrm {T} _ { (\mathrm {a b})} - \frac {1}{2} \varepsilon_ { \mathrm {A B}} \varepsilon_ { \mathrm {A ^ { \prime} B ^ { \prime}}} \mathrm {T} _ { \mathrm {C}} ^ { \mathrm {C}} \mathrm {C} ^ { \prime} + * \mathrm {T} _ { \mathrm {a b}} \\ \mathbf {\tau} = \mathbf {\tau} _ { (\mathbf {a b})} - \frac {1}{2} \mathbf {\tau} _ { \mathbf {g} _ { \mathbf {a b}}} \mathbf {\tau} _ { \mathbf {c}} ^ { \mathbf {c}} + \mathbf {i} * \mathbf {T} _ { \mathbf {a b}}. \\ \end{array}
$$
