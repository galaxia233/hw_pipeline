Problem 5.28. A fluid is "perfect" except for admitting some heat conduction, described by a heat flow 4-vector $\mathbf{q}$ . Calculate the local rate of entropy generation $\nabla \cdot \mathbf{S}$ .

## Solution

Solution 5.28. For the stress energy of the system we have
$$
\mathbf {T} = \mathbf {T} _ { \mathrm {f l u i d } } + \mathbf {T} _ { \mathrm {h e a t } } = \left[ ( \mathbf {P} + \rho ) \mathbf {u} \otimes \mathbf {u} + \mathbf {P g} \right] + \left[ \mathbf {q} \otimes \mathbf {u} + \mathbf {u} \otimes \mathbf {q} \right] .
$$
We compute the equation expressing energy conservation along the fluid flow:
$$
0 = ( \nabla \cdot \mathbf {T} ) \cdot \mathbf {u} = ( \nabla \cdot \mathbf {T} _ { \text {f l u i d } } ) \cdot \mathbf {u} + ( \nabla \cdot \mathbf {T} _ { \text {h e a t } } ) \cdot \mathbf {u } . \tag { 1 }
$$
By a short calculation (see Solution 5.20) the first term is seen to be $-\mathrm{d}\rho /\mathrm{d}\tau +(d\mathfrak{n} / d\tau)(\mathbb{P} + \rho) / \mathfrak{n}$ which by the $1^{\mathrm{st}}$ law of thermodynamics equals $-\mathbf{nTds} / \mathbf{dr}$ (see Problem 5.19). The second term is
$$
( \nabla \cdot \mathbf {T} _ { \mathrm {h e a t } } ) \cdot \mathbf {u} = ( \mathbf {q} ^ { \mu } \mathbf {u} ^ { \nu } + \mathbf {u} ^ { \mu } \mathbf {q} ^ { \nu } ) _ { ; \nu } \mathbf {u} _ { \mu } .
$$
Since $\mathbf{q}^{\alpha}\mathbf{u}_{\alpha} = 0$ (heat flux is spacelike in comoving frame) and $\mathbf{u}^{\alpha}\mathbf{u}_{\alpha} = -1$ , which imply
$$
\begin{array}{l} 0 = \left( q ^ { \mu } u _ { \mu } \right) _ { ; \nu } = q ^ { \mu } _ { ; \nu } u _ { \mu } + q ^ { \mu } u _ { \mu ; \nu } \\ 0 = \left( \mathrm {u} ^ { \mu } \mathrm {u} _ { \mu } \right) _ { ; \nu } = 2 \mathrm {u} _ { ; \nu } ^ { \mu } \mathrm {u} _ { \mu } , \\ \end{array}
$$
the second term reduces to $-\mathbf{q} \cdot \mathbf{a} - \nabla \cdot \mathbf{q}$ . Thus Equation (1) becomes
$$
0 = - n T d s / d \tau - q \cdot a - \nabla \cdot q .
$$
Finally using the definition $\mathbf{S} = \mathbf {nsu} + \mathbf {q} / \mathbf {T }$ , we compute
$$
\begin{array}{l} \nabla \cdot \mathbf {S} = \mathbf {s} \nabla \cdot ( \mathbf {n u } ) + \mathbf {n} ( \nabla \mathbf {s} \cdot \mathbf {u } ) + \frac { 1 } { T } \nabla \cdot \mathbf {q} - \frac { \nabla T } { T ^ { 2 } } \cdot \mathbf {q} \\ = n \frac { d s } { d r } + \frac { 1 } { T } \nabla \cdot q - \frac { \nabla T \cdot q } { T ^ { 2 } } \\ = - \frac { \mathbf {q} \cdot \mathbf {a} } { \mathbf {T} } - \frac { \nabla \mathbf {T} } { \mathbf {T} ^ { 2 } } \cdot \mathbf {q } . \\ \end{array}
$$
The second term says entropy is generated by heat flow thru a temperature gradient, as in nonrelativistic thermodynamics. The first term says that it is generated by flow along an acceleration, this is a "redshift" effect arising from the fact that in a accelerated system, constant temperature is not an equilibrium state: photons from the "forward" side are blueshifted in going backward, thus carrying a net heat flux.
