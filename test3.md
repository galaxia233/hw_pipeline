在史瓦西时空中，静止质量为零的光子在赤道面（$\theta = \pi/2$）内运动。已知史瓦西度规为：$$ds^2 = -\left(1 - \frac{2M}{r}\right)dt^2 + \left(1 - \frac{2M}{r}\right)^{-1}dr^2 + r^2(d\theta^2 + \sin^2\theta d\phi^2)$$利用时空的Killing矢量定义守恒量：能量 $E = (1 - 2M/r)\dot{t}$ 和角动量 $L = r^2\dot{\phi}$（其中点号表示对仿射参数 $\lambda$ 的求导）。请利用光子测地线的条件 $g_{\mu\nu}\dot{x}^\mu\dot{x}^\nu = 0$，将径向方程写为 $\left(\frac{dr}{d\lambda}\right)^2 + V_{\text{eff}}(r) = E^2$ 的形式。由此得到的有效势 $V_{\text{eff}}(r)$ 的正确表达式是什么？

解答：
1. 基础条件与守恒量史瓦西度规在赤道面（$\theta = \pi/2 \implies d\theta = 0, \sin\theta = 1$）简化为：$$ds^2 = -\left(1 - \frac{2M}{r}\right)dt^2 + \left(1 - \frac{2M}{r}\right)^{-1}dr^2 + r^2 d\phi^2$$由于度规分量不显含 $t$ 和 $\phi$，时空存在两个 Killing 矢量 $\xi^\mu = (1,0,0,0)$ 和 $\eta^\mu = (0,0,0,1)$。沿测地线的守恒量定义为：能量： $E \equiv -g_{tt}\dot{t} = \left(1 - \frac{2M}{r}\right)\dot{t}$角动量： $L \equiv g_{\phi\phi}\dot{\phi} = r^2\dot{\phi}$其中点号表示对仿射参数 $\lambda$ 的求导（即 $\dot{x}^\mu = \frac{dx^\mu}{d\lambda}$）。由此，我们可以将 $\dot{t}$ 和 $\dot{\phi}$ 表达为径向坐标 $r$ 和守恒量的函数：$$\dot{t} = \frac{E}{1 - \frac{2M}{r}}$$$$\dot{\phi} = \frac{L}{r^2}$$2. 利用零测地线条件推导对于静止质量为零的光子，其微元弧长 $ds^2 = 0$，即满足零测地线条件 $g_{\mu\nu}\dot{x}^\mu\dot{x}^\nu = 0$。展开各分量：$$-\left(1 - \frac{2M}{r}\right)\dot{t}^2 + \left(1 - \frac{2M}{r}\right)^{-1}\dot{r}^2 + r^2\dot{\phi}^2 = 0$$将前面得到的 $\dot{t}$ 和 $\dot{\phi}$ 代入上式：$$-\left(1 - \frac{2M}{r}\right) \left[ \frac{E}{1 - \frac{2M}{r}} \right]^2 + \left(1 - \frac{2M}{r}\right)^{-1}\dot{r}^2 + r^2 \left( \frac{L}{r^2} \right)^2 = 0$$化简每一项：$$-\frac{E^2}{1 - \frac{2M}{r}} + \left(1 - \frac{2M}{r}\right)^{-1}\dot{r}^2 + \frac{L^2}{r^2} = 0$$为了消去 $\dot{r}^2$ 前面的系数，我们给整组方程同时乘以 $\left(1 - \frac{2M}{r}\right)$：$$-E^2 + \dot{r}^2 + \frac{L^2}{r^2}\left(1 - \frac{2M}{r}\right) = 0$$3. 写成标准形式并确定有效势将方程移项，让 $\dot{r}^2$（即 $\left(\frac{dr}{d\lambda}\right)^2$）留在左边：$$\left(\frac{dr}{d\lambda}\right)^2 + \frac{L^2}{r^2}\left(1 - \frac{2M}{r}\right) = E^2$$对比题目给出的标准经典力学形式：$$\left(\frac{dr}{d\lambda}\right)^2 + V_{\text{eff}}(r) = E^2$$易得光子的有效势为：$$V_{\text{eff}}(r) = \frac{L^2}{r^2}\left(1 - \frac{2M}{r}\right) = \frac{L^2}{r^2} - \frac{2ML^2}{r^3}$$
